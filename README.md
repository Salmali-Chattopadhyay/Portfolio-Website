#  Personal Portfolio Website

A modern, responsive portfolio website showcasing my skills, education, experience, and services as a Data Science and Computer Science student.

##  Preview

A clean and elegant portfolio featuring a burlywood color scheme with vibrant pink (#D8125B) accents, creating a professional yet unique visual identity.

##  Features

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Interactive UI**: Smooth animations and hover effects for enhanced user experience
- **Multi-page Navigation**: Clean navigation between Home, Services, Skills, Education, and Experience sections
- **Dynamic Typing Effect**: Animated typing text displaying multiple professional roles
- **Timeline Layout**: Visual timeline for education history with interactive elements
- **Skill Visualization**: Progress bars showing proficiency levels in various technologies
- **Service Cards**: Interactive service cards with hover effects
- **Social Media Integration**: Links to LinkedIn, GitHub, Twitter, and Instagram profiles

##  Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom styling, animations, and responsive design
- **JavaScript**: Dynamic typing effect and interactive elements
- **Font Awesome**: Icon library for professional icons
- **Google Fonts**: Custom typography (Dancing Script, Times New Roman)

##  Setup and Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio-website.git
```

2. Navigate to the project directory:
```bash
cd portfolio-website
```

3. Open `index.html` in your web browser:
```bash
# On Windows
start index.html

# On macOS
open index.html

# On Linux
xdg-open index.html
```

Or simply double-click the `index.html` file.

##  Responsive Design

The website is fully responsive and optimized for:
- Desktop (1200px and above)
- Tablet (768px - 1199px)
- Mobile (below 768px)

- Chrome (recommended)

- Opera

##  Customization

To customize this portfolio for your own use:

1. Replace personal information in HTML files
2. Update the color scheme in CSS files (search for `#D8125B` and `burlywood`)
3. Replace `main.jpg` with your own profile image
4. Update social media links in `index.html`
5. Modify skills, education, and experience content as needed
